# nodeを実行する環境

Next.js コンテナ作成
> docker compose run --rm サービス名 npm create next-app . --ts

React コンテナ作成
> docker compose run --rm サービス名 npx create-react-app . --templete typescript

コンテナ起動
> docker compose up -d
